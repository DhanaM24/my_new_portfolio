Image Storage Instructions
===========================

Place your profile image in this folder with the filename: profile.jpg

To add your image:
1. Save your profile photo as "profile.jpg" in this images folder
2. The recommended image size is at least 400x500 pixels
3. Supported formats: JPG, PNG, or WebP
4. The image will be displayed with:
   - Rounded corners (20px border-radius)
   - A subtle cyan border
   - Hover effect that lifts the image up slightly
   - Box shadow glow effect

If you want to use a different filename or format:
- Update the src attribute in index.html:
  Change: <img src="images/profile.jpg" ...
  To: <img src="images/your-filename.jpg" ...
